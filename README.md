# 🏗️ Asphalt & Paving Calculators

Free online calculators for asphalt, driveway, and paving construction projects.

## 🌐 Live Site

**→ [AsphaPro.com](https://asphapro.com)**

## ✨ Features

- **Asphalt Tonnage Calculator** — estimate materials needed for your project
- **Driveway Cost Estimator** — calculate paving costs per square foot
- **Material Volume Calculator** — convert between cubic yards and tons
- **Project Budget Planner** — full cost breakdown with labor estimates

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Responsive mobile-first design
- Schema.org structured data for SEO
- No frameworks — fast, lightweight, accessible

## 📐 Calculation Formulas

| Calculator | Formula |
|---|---|
| Asphalt Tonnage | `Area (ft²) × Depth (in) ÷ 12 × Density (lbs/ft³) ÷ 2000` |
| Hot Mix Volume | `Length × Width × Depth ÷ 27` (cubic yards) |
| Compacted Density | `145 lbs/ft³` (standard hot mix asphalt) |
